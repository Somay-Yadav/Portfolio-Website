# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website! 🚀  
This project showcases my skills, education, projects, and contact information through a clean and responsive web design.

## 📌 About The Project

This portfolio website is built to represent my journey as a **CSE student** and my learning progress in web development.

The goal of this project was to practice and improve my skills in:
- HTML structure
- CSS styling
- Responsive layouts
- Flexbox
- Animations
- Modern UI design

## ✨ Features

- 🎨 Modern gradient UI design
- 📱 Responsive design for mobile devices
- 🧑‍💻 Personal introduction section
- 🎓 Education section
- 🛠 Skills showcase
- 📩 Contact section
- ✨ Hover effects and smooth animations
- ⌨️ Typing animation effect

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🚀 How To Run

1. Clone this repository

2. Open the project folder

3. Run `index.html` in your browser
